◆ pdm_fir_flt_get()

int pdm_fir_flt_get ( const pdm_fir_filter_t *  f,
int  out_bits 
)

Retrieve output value.

May be called at any rate since it does not change the filter state. The output ranges from -(2**(out_bits-1)) to +(2**(out_bits-1)). Those values correspond to all 0 or all 1 input signal. Note that the output value may still exceed this range so caller should truncate return value on its own if necessary.
